## Install

```bash
$ docker build -t <youruser>/accelerator-filesystem
$ docker run -p 4000:4000 -d <youruser>/accelerator-filesystem
```
or

```bash
$ npm  install
```

## Test

```bash
$ npm  test
```

## Build ES5 distribution

```bash
$ npm  run build
```
